Documento de Exercício Prático – HTML e CSS

Conversão de Documento Word para HTML e CSS

Este documento foi elaborado com o objectivo de servir como base para um exercício prático. O aluno deverá converter todo o conteúdo aqui apresentado para uma página HTML, aplicando estilos CSS externos de forma correcta e organizada.

Elementos a considerar no exercício:

  • Estrutura básica de um documento HTML
  • Utilização correcta de títulos e parágrafos
  • Aplicação de estilos CSS
  • Organização do conteúdo

O documento HTML final deverá conter uma hiperligação funcional: Visitar W3Schools

Folha de Estilo Externa (CSS)

Todo o estilo visual está definido num ficheiro externo chamado estilo.css, ligado através da tag <link> na secção <head>.

Boas Práticas

  • Use nomes de ficheiros claros e em minúsculas
  • Comente o código se necessário

Alojamento no GitHub

Após concluir o exercício, criar um repositório público no GitHub e enviar os ficheiros index.html e estilo.css.

No final do exercício, entregar o link do GitHub até 12 horas de Segunda-feira.
